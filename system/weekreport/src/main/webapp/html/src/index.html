第十五周讨论主题
2018年研究生学业奖学金评审结果公示
各培养学院、全体研究生： 根据《关于开展2018年研究生学业奖学金评审工作的通知》要求，按照《浙江工商大学研究生学业奖学金管理暂行办法》的规定，经学生个人申请、学院与研工部初评、学校审定，现将2018年研究生学业奖学金评审结果进行公示(见附件) 。如有异议，请于11月7日10:00前实名向党委研究生工作部反映，电话：28877048，联系人：徐心楚。

关于补招校级职能部门研究生助管的通知
全体研究生： 根据学校有关部门对校内研究生勤工助学岗位的需求，有若干校级职能部门研究生助管岗位须进行补招工作，具体岗位名单请见附件1，有意向的研究生请尽快填写助管申请表（附件2），并直接联系所报岗位的联系人（见附件1）。
